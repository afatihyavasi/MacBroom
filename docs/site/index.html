Safe by default

Destructive actions are gated behind preview and explicit confirmation.

  • Never touches auth tokens, sessions, or memory data.
  • Every clean-up is previewed first — nothing is deleted before you confirm.
  • System-critical and expensive-to-rebuild paths are protected and whitelisted.
  • Powered by tw93/mole's safety-first cleaning engine.
